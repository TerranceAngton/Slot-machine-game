# Slot Machine Game

A command-line slot machine game built in Node.js, demonstrating JavaScript programming, user input handling, and basic game logic.

## Features

- Deposit money and place bets on up to 3 lines
- Spin a virtual slot machine with randomized symbols
- Calculate winnings based on matching symbols
- Play again until the balance runs out

## Tech Stack

- Node.js
- JavaScript
- prompt-sync (for user input)
